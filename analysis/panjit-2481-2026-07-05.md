## 一、公司概況

| 項目 | 內容 |
|------|------|
| **產業** | 功率半導體 IDM（MOSFET、二極體、SiC、TVS 等） |
| **全球排名** | 低功率整流器 Top 5（Omdia 2024） |
| **核心應用** | 車用電子（~35–40%）、AI 伺服器電源、工控綠能 |
| **2025 合併營收** | 約 **131 億元** |
| **2026 目標** | 營收 **>300 億元**、市值 **>1,000 億元** |

### 三大成長引擎

1. **車用電子**：Tier 1 轉單（安世/Nexperia 事件）、電動化推升每車 MOSFET 用量
2. **AI 伺服器電源**：CSP 供應鏈、48V→800V HVDC 架構升級、高功率 MOSFET
3. **轉單 + 漲價**：非紅供應鏈、產品漲價機制、毛利率提升至 ~33%

## 七、與 AI / 機器人 thesis 的關係

| 論點 | 強茂受益程度 |
|------|-------------|
| AI 伺服器電力需求 | ✅ **直接**（MOSFET、電源管理） |
| 機器人 / 邊緣 AI | ✅ **中長期**（工控 & 綠能 13%、馬達/散熱） |
| HBM / 記憶體 supercycle | ⚠️ **間接**（伺服器電源，非核心） |
| 非紅供應鏈 | ✅ **直接**（轉單） |

強茂是 **「AI 電力基建」** 受益者，不是 **「AI 算力/memory」** 核心標的。

## 八、情境推演

### 多頭情境（168 元守住）

```
168 守穩 → 200 元 → 215（ATH）→ 突破後看 230~250

### 空頭情境（168 元失守）

```
跌破 168 → 148（MA50）→ 135 → 117（-43%）

- 功率半導體景氣反轉
- AI capex 下修連帶伺服器電源需求放緩
- 獲利不如預期，P/E 壓縮
