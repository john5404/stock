# SK 海力士長約「無價格上限」分析

> 資料來源：TrendForce（2026-07-02）、Green Economy News 引述、UBS / HSBC 研報摘要

---

## 發生了什麼？

| 項目 | SK 海力士 | 美光（對照） |
|------|-----------|-------------|
| 長約 **價格上限** | **取消**（No Price Cap） | SCA 設 **上限**，綁 Q2 2026 市價 |
| 價格下限 | 仍有保障機制 | 有 **floor + ceiling** |
| 合約年限 | 延長至 **3～5 年** | 戰略客戶長約 |
| 預付款 | 據報要求 **10～30%** 訂金 | 有綁量承諾 |
| HBM4 新產品 | 另議價 | HBM / DDR6 等 **單獨談** |

**重點**：海力士長約中，現貨/市場大漲可 **完整反映** 進合約價，不再被天花板卡住。

---

## 為什麼是結構性利多？

### 1. 定價權從買方轉向賣方

**傳統邏輯**：
```
客戶：「給你長單，但價格不能超過 X」 → 廠商犧牲 upside 換量
```

**海力士新邏輯**：
```
客戶：「怕缺貨，願意長約 + 預付」 → 廠商：量也鎖了，價也不封頂
```

供不應求時才辦得到 → 本身即說明 **HBM/DRAM 極缺**。

### 2. 呼應記憶體結構論述

| 論點 | 無上限合約如何呼應 |
|------|-------------------|
| HBM 三年難大增產 | 客戶願意 **3～5 年長約 + 預付** 搶產能 |
| AI 仍屬早中期 | 下游寧可放開價格上限也要 **鎖供給** |
| 弱化週期股属性 | **合約結構改變** = 毛利/營收能見度提高 |

**2027 HBM4 年約**：TrendForce 指三巨頭可能 **大幅調漲**，市場有 **翻倍** 討論；UBS 曾估 2027 HBM ASP **YoY +36%**。

### 3. 長約 + 無上限 = 降低「週期頂突然崩價」機率

- 有 **floor**：景氣差時仍有底價保護
- 無 **ceiling**：景氣好時 **完整吃 supercycle**
- **3～5 年**：不是一年一談就翻桌

---

## 注意事項（勿過度解讀）

| 項目 | 說明 |
|------|------|
| **海力士領先，非三巨頭一致** | 報導稱海力士可能是唯一完全不設 cap 的大廠；美光仍用 price cap |
| **2026 HBM 合約曾偏低** | 年約機制使 2026 合約價未完全跟上現貨；2027 談判意在 **補回** |
| **無上限 ≠ 永遠只漲** | 2028 供給上來時仍可能重新談降 |
| **對下游是成本利空** | NVDA / 雲端 BOM 上升 → 長期可能反壓 capex |

---

## 對美光（MU）的影響

| 面向 | 影響 |
|------|------|
| **產業定價** | 🟢 海力士抬價 → 2027 HBM4 談判基調上修，MU 間接受益 |
| **MU 合約結構** | 🟡 仍有 price cap → 短期毛利 capture 可能不如海力士 |
| **HBM 新世代** | 🟢 HBM4 單獨議價 → 新產品線仍有大幅漲價空間 |
| **敘事** | 🟢 強化 supercycle 延長至 2027+ |

## 結構定位：MU 是「過渡帶」而非純週期或純成長

```
純週期股 ←————————●——————→ 結構成長股
              美光 2026～2028
              （過渡帶）
```

| 時期 | 狀態 |
|------|------|
| **2026～2028** | 「週期性減弱、超級循環延長」 |
| **2028+** | 若 HBM 長約穩定 + 機器人起量 → 有機會 partial re-rating |
| **長期** | 除非 HBM/企業記憶體 >70% 營收且毛利穩定，否則難完全擺脫週期 |
